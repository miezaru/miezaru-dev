Configuring Git

  1. git config --global user.name "User Name"

  2. git config --global user.email mail@gmail.com

  3. git config --global core.editor "code --wait"

  4. git config --global credential.helper cache

    • Password memorisation (so that it does not ask every time when sending changes via HTTPS URL).

  5. git config --list --show-origin

    • To see all the settings that have been set and where they are set.

  6. git config --global alias.lg1 "log --graph --abbrev-commit --decorate --format=format:'%C(bold blue)%h%C(reset) - %C(bold green)(%ar)%C(reset) %C(white)%s%C(reset) %C(dim white)- %an%C(reset)%C(bold yellow)%d%C(reset)' --all"

    • Better git log (lg1).

  7. git config --global alias.lg2 "log --graph --abbrev-commit --decorate --format=format:'%C(bold blue)%h%C(reset) - %C(bold cyan)%aD%C(reset) %C(bold green)(%ar)%C(reset)%C(bold yellow)%d%C(reset)%n'' %C(white)%s%C(reset) %C(dim white)- %an%C(reset)' --all"

    • Better git log (lg2).

  8. git help.
