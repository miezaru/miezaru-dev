About colors

RGB Model: Every color can be represented by a combination of red, green and blue.

Each of the 3 base colors can take a value between 0 and 255, which leads to 16.8 million different colors

«Young guys can be messy rascals»

Color keywords:

  • 148 named colors in CSS

  • transparent is a fully transparent color. It is also the initial value of background-color

  • currentColor is the contextual computed dynamic value of the color property. If you have a text color of red and then set the border-color to be currentColor, it will also be red. If the element that you define currentColor on doesn't have a value for color defined, currentColor will be computed by the cascade instead

  • System colors

Defining colors in CSS

RGB/RGBA Notation

  • Regular RGB model, and with transparency

                                
color: rgb(0 255 255);
color: rgba(0 255 255 / 0.3)

HEXADECIMAL Notation

  • Instead of using scale from 0 to 255, we go from 0 to ff (255 in hexadecimal numbers) 

                                
color: #00ffff;

  • Shorthand, when all colors are identical pairs 

                                
color: #0ff;

  • Also have transparency

HSL

  • Hue saturation lightness / alpha

                                
color: hsl(315deg 40% 60%);
color: hsla(359deg 70% 50% / 0.5);

Shades of grey

When colors in all 3 channels are the same, we get a grey color

There are 256 pure grays to choose from
