Setting up WSL2

Read the official installation

  • Microsoft WSL2

Configuring WSL2 on a different drive

  1. Open terminal: wsl --shutdown

  2. Go to RegEdit: "Computer\HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\Lxss\"

  3. Check "BasePath" and find your wsl drive, then copy it and put where u want

  4. Then change "BasePath" to a new location

  5. Change "DistributionName" if needed

Further installation

  1. Installing wsl2 on another disc

  2. Modify the json file of the terminal app

  3. Install the Remote Development extension in VSCode

  4. Run: "sudo apt update && sudo apt upgrade" in wsl2 terminal

  5. connect wsl2 to vscode via console with the command "code ."

  6. Git setup

  7. nvm/nodejs/npm

Optional

  • Add Windows drive references to the working WSL2 directory (to avoid typing "/mnt/c/...").

    • "cd ~"

    • " ln -s /mnt/c/Users/user win_home"
